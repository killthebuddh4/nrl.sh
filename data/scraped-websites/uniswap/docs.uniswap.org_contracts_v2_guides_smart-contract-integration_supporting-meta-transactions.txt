This obviates the need for a blocking approve transaction before programmatic interactions with pool tokens can occur.ERC-712In vanilla ERC-20 token contracts, owners may only register approvals by directly calling a function which uses msg.sender to permission itself. With meta-approvals, ownership and permissioning are derived from a signature passed into the function by the caller (sometimes referred to as the relayer).
