Use ethers.js or web3.js to connect your website to Ethereum. Users will need a web3-enabled browser. On desktop this means using the MetaMask extension or something similar. On mobile, web3-compatible browsers include Trust Wallet and Coinbase Wallet. See ethereum.org to learn more.Factory ContractThe Uniswap factory contract can be used to create exchange contracts for any ERC20 token that does not already have one. factory contract can be used to find the ERC20 token address associated with an exchange contract.
